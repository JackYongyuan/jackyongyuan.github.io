CSS是什么？怎么工作的？

整理笔记时的记录，对 css 的起源，工作做更进一步更细致的记录。

CSS 是什么

摘自W3C 对 css 的描述：https://www.w3.org/TR/CSS/#intro
Cascading Style Sheets (CSS) cascading style sheets 翻译过来就是：级联样式表 CSS
CSS 是一种样式表达语言，用于给网站的编辑者对结构化标签（HTML & XML）添加样式展示给观看者。CSS是在近些年来随着人类对计算机大力发展，对电脑配件不断升级之后，人类对可视化的网页有了更多的需求，而 CSS 就是利用浏览器甚至是显示器，显卡等硬件软件，来加强色彩表现的语言。
在早期的css常常是这样的：

每个font代码起到了改变一段字体的颜色，在页面内容尚少时，维护起来还算简单。而如今人们对页面表现有了非常巨大的需求，从内容，交互，展示，听读等多方面的需求，因此上面这种代码就被舍弃了！无他：开发者维护起来太累！厂商维护成本太高！

现在的CSS都是这样的了

这样的代码不仅可以直接作用在标签上，还可以在页面内通过<style>样式内容</style>来维护，更可以通过link标签来引用外部的CSS文件。维护起来可方便了
上面说到CSS在早期是为了结构化标签提供样式表，那么 html & xml 都是可以使用 css 的。不过随着科技的发展，xml 更会被作为数据传输的一种数据格式，而html与css的配合则达到了极致。现在的css都是作用在html上。

CSS 是怎么工作的

摘自：MDN
1、浏览器载入html
2、将html文件转化成一个DOM（Document Object Model）DOM是计算机内存中的表现形式
3、浏览器会拉去该HTML相关的资源，例如图片，视频，css样式表
4、浏览器拉去到css之后会进行解析，根据不同的选择器（element，class，id等）把他们分到不同的“桶中”，浏览器基于它找到的不同的选择器，将不同的规则应用在对应的DOM的节点中并添加节点依赖的样式（这个中间步骤称为渲染树）
5、上述规则应用于渲染树后，渲染树会依照应该出现的结构进行布局
6、网页展示在屏幕上（这一步称为着色）

https://developer.mozilla.org/zh-CN/docs/Learn/CSS/First_steps/CSS%E5%A6%82%E4%BD%95%E8%BF%90%E8%A1%8C

W3C的CSS文档：https://www.w3.org/TR/css-2018/

总结

计算机技术在西方起源，早期的西方计算机做为军方使用。后来才进入到普罗大众生活中，大众对计算机能展示的东西有着巨大的需求，这种强力需求刺激了市场，加速了生活中计算机的硬件升级。可以说为了看到更好看的东西就是CSS & 一切展示型的语言的起源，为了更好看，更快的看到的东西，满足看到更多的东西，由于硬件不断升级CSS & HTML就会有着不断升级地表现力。
